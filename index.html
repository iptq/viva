What is viva?

viva is a responsive, HTML + CSS + JavaScript web framework. It's extremely easy to implement viva into your page! Read below to find out how.

Getting Started

To start, grab the latest version of viva from Github.

UI Demo

Below is a comprehensive list of the components offered in viva.

Buttons

Buttons are an important part of webpages, as they trigger actions. Here's what a button looks like in viva.

The code for this button looks like this: 

<button class="button button-white">Ångström</button>

Colored Buttons

Buttons can have different colors. Simply append add the class for the color to the element.

<button class="button button-blue">Angstrom</button>
<button class="button button-green">Angstrom</button>
<button class="button button-purple">Angstrom</button>
<button class="button button-yellow">Angstrom</button>
<button class="button button-red">Angstrom</button>
